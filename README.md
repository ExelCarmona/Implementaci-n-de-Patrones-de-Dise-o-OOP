# Python Design Patterns

Este repositorio contiene 5 proyectos pequeños y funcionales que demuestran 5 de los patrones de diseño orientados a objetos más comunes:

1. **Singleton** (Creacional): Asegura que una clase solo tenga una instancia.
2. **Factory Method** (Creacional): Proporciona una interfaz para crear objetos.
3. **Observer** (De comportamiento): Mecanismo de suscripción para notificar a objetos de eventos.
4. **Strategy** (De comportamiento): Encapsula algoritmos y los hace intercambiables.
5. **Decorator** (Estructural): Añade funcionalidades a objetos usando encapsuladores.

## Ejecución
Puedes entrar en cualquiera de las 5 carpetas y ejecutar el archivo `main.py` para ver el resultado en la consola.
